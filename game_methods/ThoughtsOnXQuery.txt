Wie ändert man mittels Xquery nur einen kleinen Teil eines xml documents? Wie werden die Argumente übergeben?
Gibt es einen "einfachen" Weg, um Zufallszahlen zu generieren, oder muss dafür eine Xquery Funktion erstellt werden?
Meine beste Vermutung, ist Funktionen zu erstellen, welche als Rückgabe einen String liefern, und diese zu verwenden
um Teile des XMLs zu editieren (bzw zu ersetzen). Bis jetzt sind es primär Funktionen, welche neuen Inhalt erzeugen, zb Um das Spiel
zu initialisieren bzw zu zurückzusetzen.



Hilfsfunktionen

declare function local:CollumnToPositionX ($c as xs:integer?)
as xs:decimal?
{
	return ($c * 21) + 10 
};
declare function local:RowToPositionY ($r as xs:integer?)
as xs:decimal?
{
	return ($c * 21) + 10
};

wähle Karte in in Spalte c Zeile r

for $c in doc("game_states1.xml")/game/cards
where $c/position_x = local:CollumnToPositionX(c) and $c/position_y = local:CollumnToPositionY(r)

Das Problem bleibt momentan die zufällige Verteilung der Karten. Man könnte natürich eine rng funktion schreiben, aber es
wäre schon, wenn es hierfür bereits Funktionen bereitstehen würden (vor allem, da eine gute rng Funktion in XQuery voraussichtlich
eine ganze Menge an Code benötigt, welcher sonst wohl nie mehr gebraucht wird. ZB Initialisierung mittels aktueller Zeit erfordert entweder
den Zugriff auf eine Uhr, oder aber eine Zeitvariable im XML file welche regelmäßig aktualisiert wird.)
