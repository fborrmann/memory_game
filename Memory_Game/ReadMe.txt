Vor dem Start des Spiels bitte folgende Schritte beachten:

- Ordner XSLT in den BaseX\webapp Ordner kopieren
- Ordner data und xsltforms in den BaseX\webapp\static Ordner kopieren
- wenn BaseX nicht unter dem Pfad C:\Program Files (x86) installiert ist:
  in methodsGame.xqm und data\svg_creator.xsl nach diesem Pfad suchen und 
  entsprechend anpassen
- Datenbanken XSLT, XSLT_highscores und XSLT_members mit den entsprechenden 
  XML Files im XSLT Ordner in BaseX anlegen
- Empfohlener Webbrowser für dieses Spiel ist Chrome
